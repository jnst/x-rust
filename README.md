# x-rust

## Run

```sh
cargo run
```

```sh
cargo test
```

## Development

```sh
cargo fmt
cargo clippy
```

## Project settings

### start the project

```sh
cargo init
```

```sh
cargo --lib init
```

### Optional settings

```sh
cargo install cargo-edit
cargo install cargo-watch
```
